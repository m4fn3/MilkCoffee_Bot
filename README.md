# MilkCoffee
MilkChocoOnline-FPSのプレイヤーのためのDiscordBOTです。

[公式サイト](https://milkcoffee.cf) | [公式サーバー](https://milkcoffee.cf/links/server)


## 機能
公式の更新情報の通知
装飾シミュレータ
グローバルチャット
等...

## 導入方法
[こちらの招待リンク](https://milkcoffee.cf/links/invite) を使用してあなたのサーバーに追加できます。
